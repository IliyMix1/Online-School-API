# Online School API

Backend API для образовательного онлайн-сервиса. Проект моделирует базовую работу онлайн-школы: регистрацию учеников, покупку курсов, просмотр уроков и домашних заданий, сдачу домашних работ и админское управление материалами.

## Stack
- Python
- FastAPI
- PostgreSQL
- SQLAlchemy Async
- Pydantic
- JWT
- bcrypt

## Возможности
- Регистрация и авторизация пользователей
- Разделение ролей student/admin
- Покупка курсов учеников
- Просмотр доступных(приобретённых) курсов
- Просмотр уроков купленного курса
- Сдача домашних работ купленного курса
- Админское создание домашних работ
- Связь домашних работ с задачами
- Связь домашних работ с уроками

## Структура БД
Основные таблицы:
- users
- students
- courses
- enrollments
- lessons
- homeworks
- tasks
- homework_tasks
- submissions
- attendance

## Запуск
1. Клонировать репозиторий
2. Создать .env
3. Указать переменные окружения
`env` 
`DATABASE_URL=...`
`SECRET_KEY=...`
`ALGORITHM=HS256`
4. Установить зависимости
5. Запустить приложение
`uvicorn main:app --reload`
